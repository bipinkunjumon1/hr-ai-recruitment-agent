# HR AI Recruitment Agent

AI-powered recruitment platform built using FastAPI, Streamlit, Groq LLM, Gmail API and Google Calendar.

## Features

- Job Description Parsing
- Resume Parsing
- ATS Candidate Ranking
- Interview Scheduling
- Google Meet Generation
- JWT Authentication
- Gmail Reschedule Detection
- HR Approval Workflow

## Tech Stack

- FastAPI
- Streamlit
- Groq
- Google Calendar API
- Gmail API
- Python
- LangGraph

## Run Backend

uvicorn main:app --reload

## Run Frontend

streamlit run frontend/app.py
